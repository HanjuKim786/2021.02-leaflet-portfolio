버스 버튼 설명
서울 내 원하는 지점으로 이동하여 적당히 확대하새요. (예> 지하철역)
버스버튼을 누르고 살짝 맵을 이동시켜 줍니다.
현재 보고있는 위치 인근의 버스 정류소가 작은 원으로 표시됩니다.
정류소를 클릭하면 정류소의 이름과 간략한 정보가 나타납니다.
해당 정류소에서 음식을 받기 버튼을 누르면 해당 정류소 위에 마커가 나타납니다.
정류소의 위치는 실제로 존재하는 것들이며 데이터는 외부에서 받아와 실시간 반영됩니다.
